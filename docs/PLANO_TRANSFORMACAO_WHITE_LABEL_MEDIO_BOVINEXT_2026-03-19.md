# PLANO DE TRANSFORMACAO - BOVINEXT WHITE-LABEL MEDIO

Data: 2026-03-19
Responsavel: Caio
Projeto: BoviNext
Objetivo: transformar o ativo atual em um produto white-label medio, vendavel na Kiwify e facil de adaptar por nicho.

## Decisao Executiva

BoviNext nao e Finnextho.

BoviNext nao e um template cru.

BoviNext e uma base SaaS white-label media, com cara de produto serio, docs claras e narrativa comercial honesta.

O nicho de pecuaria continua existindo, mas apenas como demo vertical e prova de valor.

## Oferta Oficial

Nome de categoria:

- `base SaaS white-label com IA`

Hero sugerido:

- dashboard, auth, Supabase e IA basica prontos para rebranding rapido

Subtitulo sugerido:

- uma base media para vender software por nicho sem comecar do zero

## O Que O Comprador Recebe

- codigo-fonte da base
- autenticacao e dashboard
- integracao com Supabase
- camada basica de IA ou assistente
- docs de setup, deploy e rebranding
- demo vertical de pecuaria
- estrutura comercial para revenda na Kiwify

## O Que O Comprador Precisa Configurar

- nome e identidade visual da propria marca
- dominio e ambiente de deploy
- projeto Supabase proprio
- credenciais e chaves de terceiros
- copy final da oferta
- adaptacao do demo para o nicho escolhido

## Escopo Oficial Do Produto Medio

### Entra No Pacote Base

- home ou landing institucional
- fluxo de autenticacao completo
- shell de aplicacao
- dashboard principal
- perfil, configuracoes e estados vazios
- modulos genericos reaproveitaveis
- registros, atividades, relatorios e assistente IA como camada base
- guia de setup, deploy e rebranding
- docs de uso comercial

### Nao Entra No Pacote Base

- ERP completo
- fiscal
- multiempresa pesado
- automacao profunda de nicho
- CRM monstro
- app mobile nativo
- personalizacao sob demanda ilimitada
- suporte infinito
- promessa de servir qualquer nicho sem adaptacao

## Precificacao Sugerida

| Faixa | Uso |
| --- | --- |
| `R$ 697` | lancamento agressivo para validar demanda |
| `R$ 1.497` | preco cheio alvo da oferta principal |
| `R$ 1.997` | teto sem prova comercial mais forte |

## Estrutura Da Oferta

1. abertura com promessa clara de base white-label media
2. prova visual com dashboard, auth e demo vertical
3. explicacao do que o comprador recebe
4. explicacao do que o comprador precisa configurar
5. lista clara do que nao entra
6. bonus de docs e rebranding
7. upsells e FAQ

## Upsells Recomendados

- instalacao assistida: `R$ 497` a `R$ 997`
- rebranding completo: `R$ 997` a `R$ 2.500`
- adaptacao por nicho: `R$ 1.500` a `R$ 4.000`
- suporte de arranque por 30 dias: `R$ 297` a `R$ 597`

## Narrativa Comercial

Use frases assim:

- base pronta para adaptar
- produto medio, nao promessa inflada
- white-label com caminho claro de rebranding
- demo de pecuaria como exemplo, nao como limite

Nao use frases assim:

- serve para qualquer nicho sem ajuste
- pronto para operar sem configuracao
- substitui um ERP completo
- suporte vitalicio incluso

## Frentes De Execucao

### Frente 1 - Produto

- consolidar escopo oficial
- separar core de demo vertical
- deixar a experiencia principal clara e simples

### Frente 2 - Embalagem

- melhorar README e docs canonicos
- explicitar limites, entregas e dependencias
- deixar a oferta legivel em menos de 2 minutos

### Frente 3 - Oferta

- ajustar pricing
- definir bonuses
- desenhar upsells sem exagero

## Criterio De Aceite

So considerar a primeira versao pronta para vender quando:

- a home vender a tese white-label
- auth e dashboard parecerem produto serio
- os docs explicarem o que entra e o que nao entra
- o comprador entender o que precisa configurar
- a demo de pecuaria estiver isolada como vertical
- a precificacao fizer sentido para um produto medio

## Gate Final

Se houver duvida entre:

- parecer grande e ficar vendavel
- largura e clareza
- demo bonita e produto de verdade

escolha:

- ficar vendavel
- clareza
- produto de verdade

Esse e o norte comercial do BoviNext.

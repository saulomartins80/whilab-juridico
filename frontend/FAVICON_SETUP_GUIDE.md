# Guia rapido de favicon - WhiLab

## Objetivo

Garantir que a marca `WhiLab` apareca corretamente em navegador, compartilhamento social e indexacao.

## Arquivos recomendados

- `public/favicon.ico`
- `public/favicon-16x16.png`
- `public/favicon-32x32.png`
- `public/apple-touch-icon.png`
- `public/icon-192.png`
- `public/icon-512.png`
- `public/og-image.jpg`

## Fonte visual

- Use o simbolo oficial da marca em fundo limpo.
- Evite texto pequeno dentro do favicon.
- Para `og-image.jpg`, prefira uma composicao com logo, nome `WhiLab` e um frame forte do produto.

## Dominio canonico

- Search Console: `https://whilab.com.br`
- Sitemap: `https://whilab.com.br/sitemap.xml`

## Checklist

- favicon publicado em `public/`
- `manifest.webmanifest` revisado
- `robots.txt` revisado
- Search Console configurado para `whilab.com.br`
